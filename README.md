# Агент

### В данном проекте представлен агент, который может работать с Python, SQL и RAG.

В качестве LLM используются локальный модели через Ollama.
Хранилище - ChromaDB.

### Запуск: docker compose up -d
### UI: http://localhost:8501

**Важно:** после запуска сервиса в ollama начинается фоновый процесс скачивания моделей (~2.5 Гб). Пока они не скачаются, в UI будет ошибка при запросе.
